# Olia Family Historical & Genealogy Archives

A premium, interactive, and responsive historical website dedicated to the **Olia Family** of Rander, Gujarat. The site chronicles the family's deep roots tracing back to the Central Asian steppes, their integration into the Surti Sunni Bohra mercantile community, their migration routes through Burma (Myanmar) and Pakistan, and detailed genetic and demographic records.

## Project Structure

The project is structured as a lightweight, clean, and highly performant static web application using semantic HTML5, custom CSS3 styles, and modern vanilla JavaScript:

```
Olia family/
├── index.html        # Page structure, navigation layout, and complete unabridged text copy
├── styles.css        # Premium typography, color systems, glassmorphism, responsive grids, and visual family tree connectors
├── app.js            # Demographic records database, search/filtering algorithms, modal overlay logic, and navigation hooks
└── README.md         # Project documentation (this file)
```

## Design System

The visual design is constructed with premium aesthetics to present a professional, historical feel:
- **Heritage Palette**: Deep slate background (`#050808` to `#0d1e1a`), rich gold accents (`#c5a880`), and deep emerald green highlights (`#0c3e2f`) representing Islamic scholarship and mercantile wealth.
- **Typography**: Beautiful Google Fonts. Serif headers use **Cinzel** for a classical, historical appearance; body text uses **Plus Jakarta Sans** for readability and crisp, modern UI layouts.
- **Micro-interactions**: Subtle CSS animations for fade-ins, hover elevations, and responsive navigation anchors.
- **Glassmorphism panels**: Cards utilize a transparent glassmorphism layout with gold borders to structure information clearly without cluttering the deep heritage backgrounds.

## Interactive Features

1. **Hierarchical Family Tree**: Built using semantic CSS structures to connect descendants in a logical layout. Family members are color-coded by branch (Blue for Hashimi, Emerald for Ismaili) and connected visually.
2. **Interactive Demographic Modal**: Clicking on any node in the family tree or card in the directory launches a pop-up detail card containing:
   - Full Name
   - Birth & Death Dates
   - Birth & Death Locations (Rander, Burma, Mauritius, Karachi)
   - Detailed Biographical context
3. **Genealogy Directory Engine**: A real-time search engine with keyword indexing, allowing users to filter descendants by name, location, dates, or specific branches.
4. **Migration Timeline**: A vertical timeline showing the chronological milestones of the family from pre-medieval Central Asia to the post-Partition era.

## How to Run the Site

Since the website is built using standard, zero-dependency HTML, CSS, and JS:
1. Double-click the `index.html` file to open it in any web browser.
2. Alternatively, serve it via any lightweight local server (e.g. VS Code Live Server, python http.server, or node-static).

## Historical Source References

All names, dates, and accounts are sourced from the verified historical memory of the Olia Family, including:
- **Central Asian Origins**: The Oghuz Turkic Bayat Tribe migration from Yangikent to Neyshabur, the war with the Mongols (death of Genghis Khan's son-in-law Toghuchar), the subsequent flight to the Delhi Sultanate, and resettlement in Rander, Surat following Alauddin Khilji's conquest of Gujarat in the 14th century.
- **Genetic Records**: Y-DNA matching between the Olia family and the Bhayat family of Kholvad, confirming Oghuz Turkic roots.
- **Community Context**: Surti Sunni Bohras conversion from Tayyibi Isma'ilism, Kufan/Hadrami Arab historical migration roots, and maritime trading networks (Ceylon, Maldives, Mauritius).
- **WWII Burma Evacuation**: Yusuf Ismail Olia's career as a Burmese national football player, the 1942 Japanese invasion escape, and Aishabibi's delivery of Hashim Yusuf Olia in Rander.
- **Partition Migrations**: 1947 relocations from Rander to Karachi, Sindh, and the continuous preservation of local roots in Rander by the descendants of Kasim Ismail Olia.
